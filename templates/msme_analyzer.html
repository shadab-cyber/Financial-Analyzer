⚠️

📥 How to download GSTR-3B JSON

  1. Go to gst.gov.in and login
  2. Click Services → Returns → Returns Dashboard
  3. Select Financial Year and Return Period
  4. Click GSTR-3B tile → click Download JSON
  5. Repeat for each month (up to 12 months)

📥 Download as Excel (recommended)

  1. SBI: Net Banking → My Accounts → Statement → Excel
  2. HDFC: NetBanking → Accounts → Download Statement → Excel
  3. ICICI: Internet Banking → Accounts → Statement → Excel
  4. Axis: Internet Banking → Accounts → Statement → CSV
  5. Kotak: NetBanking → My Accounts → Statement → Excel
