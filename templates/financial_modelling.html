📊 Financial Modelling

📚 How to Use This Tool

1 Go to Screener.in

Visit screener.in and search for your desired company.

2 Download Excel Data

Click "Export to Excel" button on the company page to download the financial data.

3 Clean Up Excel File

Open the downloaded Excel file and:

  • Delete all sheets except "Data Sheet"
  • Rename "Data Sheet" to "Balance Sheet & P&L"

4 Add Cash Flow Data

Create a new sheet named "Cash Flow Data":

  • Go back to Screener.in
  • Scroll down to the Cash Flow section
  • Click to expand the cash flow table
  • Copy the entire cash flow table
  • Paste it into the "Cash Flow Data" sheet

⚠️ Important: Align Dates

Make sure the dates/years align properly with the line items after pasting Cash Flow data. Adjust if needed.

5 Save & Upload

Save your Excel file and upload it using the form below.
